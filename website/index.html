できること

静止画から、音に反応するインタラクティブな壁紙まで。Mac のデスクトップを思いのままに。

動画・GIF 壁紙

mp4 / mov / webm / GIF をそのまま壁紙に。お気に入りの映像が常にデスクトップで動きます。

音楽連動ビジュアライザ

システム音声を解析し、再生中の音楽に合わせて壁紙がリアルタイムに反応します。(macOS 13+)

豊富なエフェクト

ブラー・グリッチ・色収差・ブルーム・水の波紋・ヒートヘイズなど、10 種類以上のエフェクトを重ねがけ。

パーティクル演出

雨や雪のオーバーレイを密度・速度・風・サイズまで細かく調整。季節感のあるデスクトップに。

壁紙エディタ

レイヤー・マスク・ブラシ・エフェクトチェーンでオリジナル壁紙を制作。MP4 / GIF / .wallpaper で書き出し。

コミュニティギャラリー

世界中のクリエイターの壁紙を閲覧・ダウンロード。自分の作品を公開して評価やコメントをもらえます。

マルチモニター対応

複数ディスプレイをまたぐスパニング壁紙、ディスプレイごとの個別設定の両方に対応。

RGB ライティング連携

Razer Chroma 対応。壁紙の色をキーボードやマウスの LED にリアルタイムで反映します。

ウィジェット & スクリーンセーバー

メニューバーから即操作。アイドル時には壁紙をそのままスクリーンセーバーとして表示します。

あらゆる壁紙に対応

手持ちの素材も、生成系の壁紙も。読み込めるフォーマットは多彩です。

静止画 (JPG / PNG / HEIC) 動画 (MP4 / MOV / WEBM / AVI) GIF アニメーション シェーダ生成エフェクト Web 壁紙 音楽プレイヤー壁紙 Wallpaper Engine シーン

賢く、軽く動く

動く壁紙でもバッテリーと作業を邪魔しない。状況を読んで自動で最適化します。

パフォーマンスプリセット

Ultra / High / Balanced / 省電力 から選択。FPS (15–144) と解像度 (1–100%) を細かく調整できます。

インテリジェント自動停止

他アプリ使用中・全画面表示・高 GPU 負荷・バッテリー駆動を検知し、壁紙を自動で一時停止します。

スケジュール & プレイリスト

時間帯に応じた自動切り替えや、順次・ランダム・シャッフル再生のプレイリストに対応。

デスクトップ操作はそのまま

動く壁紙の上からアイコンやファイルをクリック可能。「デスクトップを表示」にも対応します。
